*Five Minute Review*
- Day Summary 
  + How did the day go?
  + What success did I experience?
  + What challenges did I endure?
- Day Learning 
  + What did I learn today?
  + What did I learn today about myself?
  + What did I learn today about others?
  + What do I plan to do differently or the same tomorrow?
- Maintaining and growing relationships 
  + Who did I interact with?
  + Anyone I need to update?
  + Anyone I need to thank?
  + Anyone I need to ask a question?
  + Anyone I need to share feedback with?
